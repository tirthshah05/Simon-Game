Simon Game

A simple Simon Game built using HTML, CSS, and JavaScript.
The boxes blink in a sequence, and you must remember and repeat the sequence from the beginning.
Each new level adds one more blink to the pattern, and the game ends when you select the wrong box.

* Features

1) Simple and clean UI

2) Blink-based pattern generation

3) Level progression

4) Fully built with HTML, CSS, and JavaScript

* How to Play

1) Watch the boxes blink in a sequence.

2) Remember the order.

3) Click the boxes in the same sequence.

4) Each correct round increases the level.
